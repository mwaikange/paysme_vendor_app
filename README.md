# PaySME Vendor App

This workspace contains a React Native Expo app for PaySME vendors to process invoice tokens, top up credits, track transaction history, and manage profile settings.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the Expo development server:
   ```bash
   npx expo start
   ```

3. Open the app on a simulator or mobile device using the current Expo Go app.

## SDK

This project is configured for Expo SDK 54.

## Features

- Vendor login with ID and password
- Process PaySME invoice tokens and authorize payments with a 5-digit PIN
- Prepaid vendor top-up discount and credit balance tracking
- Loan vendor loan balance, fee withdrawal, and loan repayment
- Transaction history screen
- Profile settings with password and PIN update flows
- Loan vendor application flow with document placeholders
